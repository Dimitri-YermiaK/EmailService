# Email Service

Microsserviço de envio de e-mails desenvolvido com Spring Boot, seguindo boas práticas de arquitetura de microsserviços.

## Tecnologias

- Java
- Spring Boot
- Spring Mail
- Maven

## Funcionalidades

- Envio de e-mails via API REST
- Configuração flexível de servidor SMTP
- Estrutura de microsserviço independente

## Como executar

Configure as credenciais de e-mail no arquivo `application.properties` e execute:

```bash
./mvnw spring-boot:run
```
